# 영어 단어 사전

초등학생과 어른이 함께 사용할 수 있는 무료 영어 단어 사전 웹페이지입니다.

모르는 영어 단어가 나왔을 때 아이가 스스로 검색하고, 마음에 드는 단어를 단어장에 저장한 뒤 나중에 CSV 파일로 다운로드할 수 있도록 만들었습니다.

## 주요 기능

- 영어 단어 검색
- 어린이 모드 / 어른 모드 전환
- 품사 표시
- 영영 뜻 표시
- 예문이 있을 경우에만 예문 표시
- 한글 뜻 영역 없음
- 단어장 저장
- 저장 단어 수정
- 저장 단어 삭제
- 저장 단어 CSV 다운로드
- 단어 테스트 기능
- 정답 선택 시 폭죽 효과
- 모바일 / 아이패드 홈 화면 바로가기용 UI

## 사용 대상

- 초등학교 1학년
- 영어 단어를 함께 확인하고 싶은 보호자
- 영영 뜻 중심으로 단어를 정리하고 싶은 사용자

## 어린이 모드

어린이 모드는 아이가 보고 따라 쓰기 쉽도록 글자를 크게 표시합니다.

- 단어 크게 표시
- 품사 크게 표시
- DEF. 1개 표시
- EX. 예문이 있는 경우에만 표시
- 파란 계열의 밝은 UI

## 어른 모드

어른 모드는 조금 더 사전처럼 볼 수 있도록 구성했습니다.

- 단어 표시
- 품사 표시
- 대표 DEF. 표시
- 추가 정의가 있을 경우 MORE DEF. 표시
- 예문이 있는 경우에만 표시
- 유사어와 반의어가 있을 경우 표시

## 테스트 기능

테스트는 한글 없이 영어로만 진행됩니다.

- 문제는 DEF. 형태로만 출제됩니다.
- 보기는 영어 단어 3개입니다.
- 입력한 단어 중 무작위로 최대 5문제가 나옵니다.
- 정답을 맞히면 폭죽 효과가 나옵니다.

## 단어장 기능

검색한 단어는 단어장에 저장할 수 있습니다.

저장되는 정보:

- 단어
- 품사
- 영영 뜻
- 영문 예문
- 유사어
- 반의어
- 검색일자

저장한 단어는 나중에 수정하거나 삭제할 수 있습니다.

## CSV 다운로드

저장한 단어장은 CSV 파일로 다운로드할 수 있습니다.

CSV 열 구성:

| 열 | 내용 |
|---|---|
| A열 | 단어 |
| B열 | 영영 뜻 |
| C열 | 유사어 |
| D열 | 반의어 |
| E열 | 검색일자 |

파일명 예시:

```txt
my_wordbook_2026-06-20.csv
